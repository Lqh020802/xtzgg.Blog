# 16-断网提醒功能
[AirL](https://mrhope.site)小于 1 分钟
> 

断网提醒功能很简单,使用的是 window 的`onoffline`和`ononline`事件. 用来提醒用户网络已经断开或者网络已经连接上
```html
 <h2>断网提醒功能</h2>
```

```javascript
 window.addEventListener("online", () => {
    alert("网络连接成功");
});
window.addEventListener("offline", () => {
    alert("断网了,您的网络连接有问题哦,请检查网络");
});
```
